\section{Summary}
This document details the coding conventions used throughout Reqourse. It is incredibly important that all developers adhere to these conventions, as it can easily double the readability of our code. (It's all about that Bus Factor!)
\section{Ruby}
  \subsection{Style Guidelines}
  All code must follow the community \href{https://github.com/bbatsov/ruby-style-guide}{Ruby Style Guide}. To ensure this, we will be using a gem called \href{http://rubocop.readthedocs.io/en/latest/}{Rubocop} that statically analyzes code and prints out a log of all convention violations. In the future we will incorporate this into our deployment cycle so that this gem can automatically prevent convention violations from being pushed to production.
